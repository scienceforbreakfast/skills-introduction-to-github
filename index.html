Branches are useful for creating isolated changes without affecting the main branch.

This is useful for experimenting, drafting, planning, or making any changes without fully commiting to changes to the overall project.

Let's see how we can work with branches.

Making a branch

  • Go to your repository
  • Click on "main" branch button
  • Define a new name for a branch
  • Click down below, "Create branch [your branch name] from main

That's it! You've created your first branch!

From the same branch button, you can now switch between branches that you wish to work under before choosing to commiting files or merging branches.
